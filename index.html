About ThreeD

ThreeD is a minimalist, ultra-lightweight web application designed for fast and reliable 3D model viewing. Built purely on open-source web technologies and Three.js, it operates entirely client-side without heavy bundlers or unnecessary frameworks. Drop your assets and inspect them instantly.

Keymap & Navigation

  • LMB Select Object
  • MMB Rotate View
  • RMB Drag Pan View
  • Scroll Zoom In/Out
  • RMB Click Context Menu
  • DEL Delete Active Object
